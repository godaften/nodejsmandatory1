Tools

So, so many tools... I only mention a few.

VSCode

Visual Studio Code is a streamlined code editor with support for development operations like debugging, task running, and version control. It aims to provide just the tools a developer needs for a quick code-build-debug cycle and leaves more complex workflows to fuller featured IDEs, such as Visual Studio IDE.

How to install Visual Studio Code

  • 1: Download the Windows installer from code.visualstudio.com
  • 2: Follow the prompts.

Extensions

Visual Studio Code has a lot of extensions which makes it a full blown IDE matching your deepest desires. It's easy to install the extensions from within the program.

Browser

Use a browser of your choice to view your frontend work. The development tools of the browser can greatly help with viewing and testing Javascript, manipulating html and css.

Github

Use Git and Github version control when developing. It's not up for discussion.
