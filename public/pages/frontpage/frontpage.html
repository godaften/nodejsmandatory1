So what have I done?

I've made a responsive website in NodeJS using Express and Bootstrap with 5 pages, a navbar and a footer. It's my first time using a CSS framework. I've been wanting to try Bootstrap for years, but I just haven't gotten to it. Now was a good time. It's been quite fun and something I'm gonna use again when building and testing a small app where the design doesn't need to be to custom.

I've also looked at Tailwind. It's more of a library with a lot of css classes ready to use for everything. It takes longer to build up a page with Tailwind than Bootstrap, but it's much more customizable and still much faster than plain CSS plus you get a perfect consistent class structure. Very nice.
